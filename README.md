# 💬 ChatApp

A modern real-time chat application built using the MERN stack with Socket.IO for instant messaging.

## 🚀 Live Demo

- 🌐 Frontend: https://chat-app-pied-chi-1sqa5my3w2.vercel.app
- ⚙️ Backend API: https://chat-app-fcc5.onrender.com


## ✨ Features

### 🔐 Authentication
- JWT Authentication with HTTP-Only Cookies
- User Registration & Login
- Secure Logout
- Persistent Login Sessions

### 💬 Messaging
- Real-time Messaging using Socket.IO
- Typing Indicator
- Message Read Status
- Unread Message Count
- Delete Messages
- Edit Messages
- Pin Important Messages
- Emoji Reactions
- Reply to Messages
- Notification Sound
- Auto Scroll to Latest Message

### 👥 User Features
- Online / Offline Status
- Last Seen
- User Profile Page
- Update Profile
- Profile Photo Upload (Cloudinary)
- About/Bio Section
- Block / Unblock Users

### 📁 Media Sharing
- Image Upload
- File Sharing
- Media Gallery
- Cloudinary Storage Integration

### 🎨 User Interface
- Responsive Design
- Clean & Modern Chat UI
- Sidebar with User List
- Welcome Screen
- Loading States
- Error Handling
- Mobile Friendly Layout

### ⚡ Real-Time Features
- Instant Message Delivery
- Live Online Users
- Real-time Profile Updates
- Real-time Block Status Updates
- Socket.IO Powered Communication

### 🛠️ Backend
- RESTful API
- MongoDB Atlas
- Express.js
- Mongoose ODM
- Cookie-Based Authentication
- Multer File Upload
- Cloudinary Integration

### 🚀 Deployment
- Frontend deployed on Vercel
- Backend deployed on Render
- MongoDB Atlas Database
- Automatic GitHub Deployment


## 🛠️ Tech Stack

### Frontend
- React
- Redux Toolkit
- React Router
- Axios
- Socket.IO Client
- Tailwind CSS
- DaisyUI

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Socket.IO
- JWT Authentication
- Multer
- Cloudinary

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Gautamkumar098/Chat-app.git
```

### Backend

```bash
cd Backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🔑 Environment Variables

### Backend (.env)

```env
DB_URL=your_mongodb_connection
JWT_SECRET_KEY=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
FRONTEND_URL=http://localhost:3000
```
